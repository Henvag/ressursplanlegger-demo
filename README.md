# Ressursplanlegger-demo

En enkel **ressursplanlegger-demo** med dashboard og planleggingsvisning.

## Fungerende faner

- **Dashboard**
- **Ressursplanlegger**
- **Kompetansekart**
- **Prosjektstyring**

## Inspirasjon / grunnlag

Laget med utgangspunkt i Wizrd Solutions sin demo på nettsiden:  
[wizrd.no – Ressursplanlegger](https://www.wizrd.no/ressursplanlegger)

## Bygget med

Utviklet ved hjelp av **Cursor** og **Codex**, hvor funksjonalitet og UI-interaksjoner er implementert iterativt.

## Teknologi-stack

- **HTML** (statisk layout i `index.html`)
- **CSS** (styling i `styles.css`)
- **JavaScript (vanilla)** (interaktivitet i `app.js`)

## Kjør lokalt

1. Åpne `index.html` i nettleser (dobbelklikk), eller
2. Kjør via en enkel lokal webserver (f.eks. “Live Server”) for best utviklingsflyt.
